## Immersive Favèlla (1.12.2 Backport)

![alt text](https://img.shields.io/badge/Minecraft-1.12.2-blue.svg)![alt text](https://img.shields.io/badge/License-LGPL--2.1-green.svg)

Immersive Favella is an unofficial backport of the acclaimed Immersive Melodies mod to Minecraft 1.12.2.

![Instruments](https://api-rk.conczin.net/mcr/atlas?minecraft_version=1.20.1&locations=_immersive_melodies:instruments&dependencies=https://github.com/Luke100000/ImmersiveMelodies/tree/1.20.1&resolution=48&background=fancy&row_size=8)

## Features

Just like the original mod, most features and instruments are ported and work perfectly. For more info, check [original Immersive Melodies page](https://www.curseforge.com/minecraft/mc-mods/immersive-melodies)

## Dependencies

This mod requires MixinBooter to work.
